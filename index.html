***README***

1) To write text just click on white field and type.

2) To style text just click on given button and the text will be stylized with chosen option.

3) To export file click on "export text" button and file "saved-text.JSON" will be downloaded.

4) To import file click on "choose file" button, select and choose .JSON file. After that, click "import text" button and content of .JSON file will appear in the box. The JSON file should have property "text", otherwise, there will appear 'undefined' text.
